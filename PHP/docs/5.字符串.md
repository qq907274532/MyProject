##总结  
###转义字符  
* 以单引号为定界符的php字符串，支持两个转义\'和\\  
* 以双引号为定界符的php字符串，支持下列转义： 
\n 换行（LF 或 ASCII 字符 0x0A（10））  
\r 回车（CR 或 ASCII 字符 0x0D（13））   
\t 水平制表符（HT 或 ASCII 字符 0x09（9））  
\\ 反斜线  
\$ 美元符号   
\" 双引号  
\[0-7]{1,3} 此正则表达式序列匹配一个用八进制符号表示的字符   
\x[0-9A-Fa-f]{1,2} 此正则表达式序列匹配一个用十六进制符号表示的字符
